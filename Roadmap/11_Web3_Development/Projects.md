# **Phase 11: Web3 - Mini-Projects**

Build these projects to solidify your understanding of smart contracts and DApp integration.

---

## **Project 1: Create Your Own Token (ERC-20)**
The ERC-20 standard is what most tokens (like SHIB or LINK) use.
- **Goal:** Create a token with your own name and symbol (e.g., "ZOHO Token").
- **Logic:** Implement functions to `transfer` tokens and `checkBalance`.
- **Bonus:** Use the **OpenZeppelin** library to make your token secure in 5 lines of code.

---

## **Project 2: Decentralized Voting App**
A transparent voting system where every vote is recorded on the blockchain.
- **Goal:** Users can vote for a candidate, but only once.
- **Logic:** Use a `mapping(address => bool)` to track who has voted.
- **Frontend:** Build a dashboard to see the current vote counts.

---

## **Project 3: NFT Minter**
- **Goal:** Allow users to upload an image and "Mint" it as a unique NFT.
- **Logic:** Use the ERC-721 standard.
- **Integration:** Connect to **IPFS** (InterPlanetary File System) to store the image data.
