### **HTML5 (The Structure)**
- **Semantic HTML:** Use `<header>`, `<footer>`, `<article>`, and `<nav>` instead of just `<div>`. This is great for SEO and accessibility.
- **Forms:** Deeply understand how to handle user input and validation.

### **CSS3 (The Style)**
- **Flexbox & Grid:** These are mandatory for modern responsive layouts.
- **CSS Variables:** For theming (Dark/Light mode).
- **Transitions & Animations:** For that "premium" feel.

### **Vanilla JavaScript (The Brain)**
- **DOM Manipulation:** How to change the page without refreshing.
- **Fetch API:** How to get data from a server.
- **ES6+ Features:** Arrow functions, Destructuring, and Template Literals.

## **2. Frontend Frameworks**

Zoho uses various frameworks across its 50+ apps, but the big ones are:

### **React.js**
- **Components:** Thinking in modular UI.
- **Hooks:** `useState`, `useEffect`, and custom hooks.
- **State Management:** Context API or Redux.

### **Vue.js**
- **Reactivity:** How Vue tracks changes.
- **Directives:** `v-if`, `v-for`, `v-model`.

## **4. Responsive Design**
Your apps must work on a 27-inch monitor and a 5-inch phone.
- **Media Queries:** `@media (max-width: 600px) { ... }`
- **Mobile-First Approach:** Design for small screens first.
