# **Phase 4: OS & Computer Networks**

These subjects form the "Theory" part of your technical interview.

---

## **1. Operating Systems (OS)**

- **Process vs Thread:** A process is a program in execution; a thread is a subset of a process.
- **Deadlock:** When two or more processes are stuck waiting for each other to release resources.
    - Conditions: Mutual Exclusion, Hold and Wait, No Preemption, Circular Wait.
- **Memory Management:** Paging, Segmentation, and Virtual Memory.
- **Scheduling Algorithms:** FCFS (First Come First Serve), Round Robin, SJF (Shortest Job First).

---

## **2. Computer Networks**

- **OSI Model (The 7 Layers):** 
    - Physical, Data Link, Network, Transport, Session, Presentation, Application.
- **TCP vs UDP:** 
    - **TCP:** Reliable, connection-oriented (like a phone call).
    - **UDP:** Fast, connectionless (like a letter).
- **HTTP vs HTTPS:** The 'S' stands for Secure (SSL/TLS).
- **DNS (Domain Name System):** The phonebook of the internet (translates `google.com` to an IP address).

---

## **3. Practical Knowledge**
Be able to explain what happens when you type `www.zoho.com` in your browser. (DNS -> IP -> TCP Handshake -> HTTP Request -> Server Response -> Rendering).
