# 🎬 Singu - The Raja | Episode 7 Production

Welcome to the official production repository for **Singu - The Raja**. This repository is structured as a professional animation studio to streamline the development of Episode 7 and beyond.

---

## 🏛️ Studio Architecture
We have restructured the project into specialized departments to keep assets and workflows organized:

- **📂 [00_Production_Office](./0shotss%20copy/00_Production_Office)**: Project management, schedules, and administrative files.
- **📂 [01_Script_Department](./0shotss%20copy/01_Script_Department)**: Raw scripts (PDF/DOCX), processed Markdown files, and AI prompt generation.
- **📂 [02_Art_Department](./0shotss%20copy/02_Art_Department)**: Character sheets, environment concepts, and visual assets.
- **📂 [03_Film_Studio](./0shotss%20copy/03_Film_Studio)**: Active **Shotcut** project files, timelines, and raw renders.
- **📂 [04_Tech_Lab](./0shotss%20copy/04_Tech_Lab)**: Python automation scripts for script parsing and asset management.
- **📂 [99_Archive](./0shotss%20copy/99_Archive)**: Historical versions and unused assets kept for safety.
